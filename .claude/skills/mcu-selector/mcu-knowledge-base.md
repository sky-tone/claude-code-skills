## 四、开发生态评分标准

| 维度 | ★★★★★ (5分) | ★★★☆☆ (3分) | ★☆☆☆☆ (1分) |
|------|-------------|-------------|-------------|
| **IDE** | 官方免费IDE + VS Code插件 | 官方IDE但付费，或仅Keil/IAR | 仅命令行工具链 |
| **SDK** | HAL+LL双层，代码生成器(CubeMX) | 基本HAL库，手动配置 | 仅寄存器手册 |
| **RTOS** | 官方支持FreeRTOS+Zephyr+RT-Thread | 支持1-2种主流RTOS | RTOS支持有限 |
| **社区** | GitHub星>5K, 论坛日活>1K | 中文社区活跃，资料较全 | 仅英文手册，社区小 |
| **开发板** | Nucleo/DK等低价评估板，全球可购 | 官方开发板，渠道有限 | 仅EVK，价格>500元 |
| **文档** | 中英文全套，AN/视频/培训 | 英文datasheet+RM完善 | 文档不全或仅中文 |

## 五、封装选型指南

| 封装类型 | 引脚间距 | PCB工艺要求 | 散热 | 手工焊接 | 适用场景 |
|---------|---------|-----------|------|---------|---------|

## 六、供应链风险评估矩阵

| 风险因素 | 低风险 (1分) | 中风险 (3分) | 高风险 (5分) |
|---------|-------------|-------------|-------------|
| **供应商数量** | ≥3家授权代理 | 1-2家代理 | 单一来源 |
| **交货周期** | ≤12周 | 12-26周 | >26周 |
| **生命周期** | Active + 10年承诺 | Active, 无明确承诺 | NRND/即将EOL |
| **国产替代** | Pin兼容替代≥2款 | 功能替代存在 | 无替代方案 |
| **库存水位** | 代理商现货充足 | 需预约排产 | 全球缺货 |
| **价格波动** | 年度合同价稳定 | 季度调价≤10% | 现货溢价>30% |
| **地缘风险** | 多地区生产 | 单一地区但稳定 | 受出口管制影响 |

## 八、常用 MCU 对标关系

在选型或国产替代时，以下对标关系可作为参考：

| 原型号 | 国产替代 | 兼容度 | 备注 |
|--------|---------|-------|------|
| STM32F103 | GD32F103 / CH32F103 | Pin+软件 | 注意Flash等待周期差异 |
| STM32F407 | GD32F407 / AT32F407 | Pin+部分软件 | DMA/中断优先级可能不同 |
| STM32G431 | GD32G431 / APM32F407 | Pin | 外设寄存器有差异 |
| ESP32-C3 | BL616 / W801 | 功能替代 | 软件需重写 |
| nRF52832 | PHY6222 / CH582 | 功能替代 | BLE协议栈不同 |
| MSP430 | HC32L13x / CW32 | 功能替代 | 架构不同需重写 |
| ATmega328P | LGT8F328P | Pin+软件 | Arduino兼容 |

## 九、选型报告术语表

| 术语 | 说明 |
|------|------|
| **CoreMark** | EEMBC 标准 MCU 性能基准测试 |
| **DMIPS** | Dhrystone MIPS，每秒百万条指令 |
| **μA/MHz** | 每 MHz 运行电流，功耗效率指标 |
| **AEC-Q100** | 汽车电子元器件可靠性测试标准 |
| **ASIL** | 汽车安全完整性等级 (ISO 26262) |
| **TrustZone** | ARM 硬件安全隔离技术 |
| **PSA Certified** | ARM 平台安全架构认证 |
| **Pin 兼容** | 引脚定义兼容，可直接替换 |
| **NRND** | Not Recommended for New Design，不建议新设计使用 |
| **EOL** | End of Life，停产 |
| **MOQ** | Minimum Order Quantity，最小起订量 |
| **Lead Time** | 交货周期 |
| **BOM** | Bill of Materials，物料清单 |
| **FOC** | Field-Oriented Control，磁场定向控制 |
| **BLDC** | Brushless DC Motor，无刷直流电机 |
| **QSPI/OSPI** | 四线/八线 SPI Flash 接口 |
| **LTDC** | LCD-TFT 显示控制器 |
