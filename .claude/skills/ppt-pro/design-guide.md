## 一、版式设计黄金法则

### 1.1 网格系统

```
┌──────────────────────────────────────────┐
│ margin                                   │
│  ┌────────────────────────────────────┐  │
│  │  标题区 (H: 0.8in)                 │  │
│  ├────────────────────────────────────┤  │
│  │                                    │  │
│  │  内容区 (H: ~5.2in)               │  │
│  │                                    │  │
│  │                                    │  │
│  ├────────────────────────────────────┤  │
│  │  页脚区 (H: 0.4in)                │  │
│  └────────────────────────────────────┘  │
│                                          │
└──────────────────────────────────────────┘

尺寸标准 (16:9 宽屏):
- 幻灯片: 13.333 × 7.5 英寸
- 四周边距: 0.5 ~ 0.7 英寸
- 内容安全区: 12.333 × 6.3 英寸
```

### 1.2 留白比例

| 页面类型 | 建议留白率 | 说明 |
|---------|-----------|------|
| 封面页 | 50-60% | 仅标题+少量文字 |
| 章节页 | 45-55% | 大数字+标题 |
| 内容页 | 30-40% | 要点不超过6条 |
| 数据页 | 35-45% | 大数字留呼吸感 |
| 图表页 | 25-35% | 图表为主 |

### 1.3 对齐规则

- **强左对齐**：标题、正文、列表统一左对齐线
- **顶部基线对齐**：同行元素顶部对齐
- **等间距分布**：卡片、图标间距一致
- **中轴对齐**：封面、结束页标题居中

## 二、字体规范

### 2.1 中文 PPT 推荐字体

| 用途 | 字体选择 | 字号 | 字重 |
|------|---------|------|------|
| 大标题 | 微软雅黑 / 思源黑体 Bold | 36-48pt | Bold |
| 章节标题 | 微软雅黑 / 思源黑体 | 28-36pt | Bold |
| 页面标题 | 微软雅黑 | 24-28pt | Bold |
| 正文 | 微软雅黑 Light | 14-18pt | Regular |
| 说明/注释 | 微软雅黑 Light | 10-12pt | Light |
| 数据数字 | Arial / DIN | 48-72pt | Bold |

### 2.2 英文 PPT 推荐字体

| 用途 | 字体选择 | 字号 |
|------|---------|------|
| 标题 | Segoe UI Semibold / Montserrat Bold | 28-44pt |
| 正文 | Segoe UI / Open Sans | 14-18pt |
| 数据 | Roboto Mono / DIN Condensed | 48-72pt |
| 等宽 | Consolas / Fira Code | 12-14pt |

### 2.3 行间距

| 字号范围 | 行间距倍数 | 段间距 |
|---------|-----------|--------|
| ≥ 28pt | 1.2x | 12-16pt |
| 16-24pt | 1.3x | 8-12pt |
| 12-16pt | 1.4x | 6-8pt |
| ≤ 12pt | 1.5x | 4-6pt |

## 三、配色规范

### 3.1 配色体系

每套主题包含 8 个色值：

```
┌─────────────────────────────────────────┐
│  PRIMARY     ████  主色（标题、重点线条）    │
│  SECONDARY   ████  辅色（次级信息、图标）    │
│  ACCENT      ████  点缀色（CTA、高亮数字）  │
│  TEXT_DARK   ████  深色文字                │
│  TEXT_LIGHT  ████  浅色文字（深底上使用）    │
│  BG_DARK     ████  深色背景（封面、章节页）  │
│  BG_LIGHT    ████  浅灰背景（卡片、色块）   │
│  BG_WHITE    ████  白色背景（内容页主背景）  │
└─────────────────────────────────────────┘
```

### 3.2 色彩使用规则

1. **60-30-10 法则**：
   - 60% 基底色（BG_WHITE / BG_DARK）
   - 30% 主色（PRIMARY / SECONDARY）
   - 10% 点缀色（ACCENT）

2. **对比度要求**：
   - 白底上文字色值 ≤ `#666666`
   - 深底上文字色值 ≥ `#CCCCCC`
   - WCAG AA 标准：对比度 ≥ 4.5:1

3. **配色一致性**：
   - 全篇不超过 4 种颜色
   - 同类元素使用同一色值
   - 页面间配色风格统一

## 五、内容写作规范

### 5.1 标题写作

| 原则 | ❌ 差的标题 | ✅ 好的标题 |
|------|-----------|-----------|
| 动作化 | "项目进展" | "项目提前2周交付" |
| 具体化 | "销售数据" | "Q3 销售额增长 47%" |
| 结论化 | "市场分析" | "Z世代成为核心增长引擎" |
| 简洁化 | "关于我们公司今年的发展方向" | "2026 战略三大支柱" |

### 5.2 要点写作

- 每条要点 ≤ 20 个中文字（≤ 15 个英文词）
- 使用平行结构（动词开头 or 名词开头，统一）
- 从最重要到最不重要排列
- 避免完整句子，使用短语

### 5.3 数字展示

- 大数字使用 48-72pt，加粗
- 添加单位和上下文（"↑15% vs 去年"）
- 正面数据用 ACCENT 色，负面数据用红色
- 最多同时展示 4 个关键指标

## 六、python-pptx 技术备忘

### 6.1 坐标系统
```python
# 1 英寸 = 914400 EMU
# 1 厘米 = 360000 EMU
# 1 磅(pt) = 12700 EMU

from pptx.util import Inches, Cm, Pt, Emu

# 16:9 标准尺寸
SLIDE_WIDTH  = Inches(13.333)  # = Emu(12192000)
SLIDE_HEIGHT = Inches(7.5)     # = Emu(6858000)
```

### 6.2 常用形状类型
```python
from pptx.enum.shapes import MSO_SHAPE

MSO_SHAPE.RECTANGLE           # 矩形
MSO_SHAPE.ROUNDED_RECTANGLE   # 圆角矩形
MSO_SHAPE.OVAL                # 椭圆/圆形
MSO_SHAPE.CHEVRON             # 箭头形
MSO_SHAPE.RIGHT_ARROW         # 右箭头
MSO_SHAPE.PENTAGON            # 五边形
```

### 6.3 文本对齐
```python
from pptx.enum.text import PP_ALIGN, MSO_ANCHOR

PP_ALIGN.LEFT      # 左对齐
PP_ALIGN.CENTER    # 居中
PP_ALIGN.RIGHT     # 右对齐

MSO_ANCHOR.TOP     # 顶部对齐
MSO_ANCHOR.MIDDLE  # 垂直居中
MSO_ANCHOR.BOTTOM  # 底部对齐
```

### 6.4 设置透明背景
```python
shape.fill.background()        # 透明填充
shape.line.fill.background()   # 无边框
```

### 6.5 添加图片
```python
slide.shapes.add_picture("image.png", left, top, width, height)
```

### 6.6 添加表格
```python
table_shape = slide.shapes.add_table(rows, cols, left, top, width, height)
table = table_shape.table
cell = table.cell(row_idx, col_idx)
cell.text = "内容"
```
